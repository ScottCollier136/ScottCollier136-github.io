FizzBuzz

FizzBuzz is the quintessential programming test.

The problem given is simple. When given an integer n, write a function that will iterate from 1 to n, writing each number out. If that number is a multiple of 3, instead write "Fizz", and if it's a multiple of 5, write "Buzz". Finally if it is a multiple of 3 and 5, write "FizzBuzz".

It is generally used to test understanding of conditionals.


                public void FizzBuzz(int n){

                    int i;

                    for(i=1; i < n+1; i++){
                        if(((i % 3) == 0) && ((i % 5) == 0)){
                            System.out.println("FizzBuzz");
                        }
                        else if((i % 3) == 0){
                            System.out.println("Fizz");
                        }
                        else if((i % 5) == 0){
                            System.out.println("Buzz");
                        }
                        else{
                            System.out.println(i);
                        }
                    }
                }

Check if Anagram

This function checks if two strings are anagrams of eachother.

This function was designed to work alongside the use of spaces and punctuation.


                public static void CheckIfAnagram(String stringOne, String stringTwo){

                    //Gets rid of punctuation and spacing
                    stringOne = stringOne.replaceAll("[^a-zA-Z ]", "").replaceAll("\\s+","");
                    stringTwo = stringTwo.replaceAll("[^a-zA-Z ]", "").replaceAll("\\s+","");

                    //Puts it all into lowercase and an array form
                    char[] arrayOne = stringOne.toLowerCase().toCharArray();
                    char[] arrayTwo = stringTwo.toLowerCase().toCharArray();

                    Arrays.sort(arrayOne);
                    Arrays.sort(arrayTwo);

                    if(Arrays.equals(arrayOne,arrayTwo)){
                        System.out.println("These two strings are anagrams");
                    }
                    else{
                        System.out.println("These two strings are NOT anagrams");
                    }
                }

Reverse a String and Check if it's a Palindrome

These functions are designed to manually reverse a string and, utilizing the reversed string, check if it's a palindrome.

These functions were designed to work alongside the use of spaces and punctuation.


                public static String ReverseString(String stringVar){

                    int i;
                    char[] tempArray = stringVar.toCharArray();
                    String reversedString = "";

                    for(i=stringVar.length()-1; i>=0; i--){
                        reversedString = reversedString + tempArray[i];
                    }

                    return reversedString;
                }

                public static void CheckIfPalindrome(String stringVar){

                    //Removes all punctuation and spaces
                    stringVar = stringVar.replaceAll("[^a-zA-Z ]", "").replaceAll("\\s+","");

                    String reverseString = ReverseString(stringVar.toLowerCase());

                    if(reverseString.equals(stringVar.toLowerCase())){
                        System.out.println("These strings are palindromes");
                    }
                    else{
                        System.out.println("These strings are Not Palindromes");
                    }
                }

Find Duplicate Numbers

This function takes in an integer array, and prints out what numbers are duplicated, and at what indexes.


                //The odd spacing of the"<>"'s is due to html not liking them next to a character, and falsely reading a new element
                public static void FindDuplicateNumbers(int[] array) {

                    int i;

                    HashMap< Integer, ArrayList< Integer >> myHash = new HashMap< Integer, ArrayList< Integer >>();


                    for(i=0; i< array.length; i++){
                        if(myHash.containsKey(array[i])) {
                            myHash.get(array[i]).add(i);
                        }
                        else{
                            myHash.put(array[i],new ArrayList< Integer >());
                            myHash.get(array[i]).add(i);
                        }
                    }

                    for(Map.Entry< Integer, ArrayList< Integer >> mapElement : myHash.entrySet() ){
                        if(mapElement.getValue().size() > 1){
                            System.out.println("The value " + mapElement.getKey() + " was duplicated at indexes " + mapElement.getValue());
                        }
                    }


                }
